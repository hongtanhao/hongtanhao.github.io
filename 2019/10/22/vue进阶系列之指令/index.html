指令

v-text

作用： 更新元素的 textContent

v-text vs Mustache 语法

v-html

作用： 操作元素中的HTML标签

v-text vs v-html

v-show

作用： 根据表达式之真假值，切换元素的 display CSS 属性

v-if

作用： 根据表达式的值的真假条件渲染元素

v-show vs v-if

v-else

限制：前一兄弟元素必须有 v-if 或 v-else-if。

v-for

作用： 基于源数据多次渲染元素或模板块

源数据更新

Vue 不能检测以下数组的变动：

Vue 不能检测对象属性的添加或删除：

v-model

作用: 实现双向数据绑定

v-on

作用： 绑定事件监听器

v-bind

作用：动态地绑定一个或多个特性，或一个组件 prop 到表达式。

v-slot

作用： 为插槽传入 prop 的时候使用

注意： v-slot 只能添加在一个 <template> 上

v-pre

作用： 跳过这个元素和它的子元素的编译过程

v-cloak

作用： 指令保持在元素上直到关联实例结束编译。

v-once

作用： 只渲染元素和组件一次。随后的重新渲染，元素/组件及其所有的子节点将被视为静态内容并跳过
